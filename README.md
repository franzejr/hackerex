#Hackerex

Simple elixir wrapper for [HackerNews API](https://github.com/HackerNews/API).
Based on [tentacat](https://github.com/edgurgel/tentacat)

## Usage

```
Hackerex.Users.find("myUser")
```

## Features

- Users
